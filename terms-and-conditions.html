Termeni și Condiții

Ultima actualizare: 4 iunie 2024

Vă mulțumim că ați ales să utilizați API-ul Dextero. Acest document conține termenii și condițiile care guvernează utilizarea serviciului nostru de API Dextero. Prin accesarea sau utilizarea API-ului nostru, sunteți de acord să respectați acești Termeni. Dacă nu sunteți de acord cu acești Termeni, vă rugăm să nu utilizați API-ul nostru.

1. Definiții

  • API: Interfața de Programare a Aplicațiilor Dextero.
  • Serviciu: Serviciul API oferit de Dextero.
  • Utilizator: Orice persoană sau entitate care accesează și utilizează API-ul Dextero.
  • Conținut: Toate datele, informațiile și materialele accesibile prin API-ul Dextero.
  • Dextero: Furnizorul serviciului API

2. Acces și Utilizare

  • Acces: Accesul la API-ul nostru este permis doar prin intermediul unei kei api înregistrată pe platforma noastră.
  • Utilizare Acceptabilă: Utilizarea API-ului trebuie să fie în conformitate cu toate legile și reglementările aplicabile. Este interzisă utilizarea API-ului pentru activități ilegale, imorale sau care încalcă drepturile altor părți.
  • Limitări de Rată: Putem impune limitări asupra numărului de solicitări pe care le puteți face API-ului nostru. Aceste limitări sunt descrise în documentația API-ului.

3. Proprietate Intelectuală

5. Confidențialitate și Date Personale

5. Securitate

  • Măsuri de Securitate: Utilizatorul trebuie să implementeze măsuri adecvate de securitate pentru a proteja datele accesate prin API.
  • Compromiterea Securității: În cazul unei încălcări a securității, Utilizatorul trebuie să informeze imediat Dextero și să colaboreze pentru a remedia problema.

Dextero își rezervă dreptul de a modifica acești Termeni în orice moment. Modificările vor fi efective de la data publicării pe site-ul nostru. Continuarea utilizării API-ului după modificări reprezintă acceptarea acestor modificări.
