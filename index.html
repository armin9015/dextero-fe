About

Our service provides a romanian dictionary, a collection of words, phrases, and their meanings. It serves as a valuable resource for language learners, writers, and anyone seeking to improve their vocabulary and understanding of language.

How to use

    
        Base url:  https://api.dextero.site/api/v1/dextero/ 
    

        
             GET | /starts-with/{letters}  
            letters - string
            Returns all the words starting with the indicated letters
    
             Example 
            Url:  https://api.dextero.site/api/v1/dextero/starts-with/arcan   
            Response: 
                {
                    "length": 6,
                    "words": [
                        "arcani",
                        "arcan",
                        "arcanul",
                        "arcana",
                        "arcane",
                        "arcanea"
                    ]
                }
            
        
    
        
             GET | /ends-with/{letters}
            letters - string
            Returns all the words ending with the indicated letters
    
             Example 
            Url:  https://api.dextero.site/api/v1/dextero/ends-with/arcan     
            Response:  
                {
                    "length": 2,
                    "words": [
                        "arcan",
                        "carcan"
                    ]
                }
            
        
    
        
             GET | /palindromes
            Returns all the palindromes from the dictionary
        
             Example 
            Url:  https://api.dextero.site/api/v1/dextero/palindromes     
            Response: 
                [
                    "alala",
                    "bab",
                    "anina",
                    "așa",
                    "axa",
                    ...
                ]
            
        
    
        
             POST | /uwu
            Body: string

            Returns the UWU text
        
             Example 
            Url:  https://api.dextero.site/api/v1/dextero/uwu     
            Body: "Unde au plecat unchiul si pisica mea uimitoare?"

            Response: 
                {
                    "sentence": "Unde au plecat unchiul si pisica mea uimitoare?"
                    "uwuSentence": "Unde auwu pwecat uwunchiuwuw zi pizica mea uwuimitoawe?  👉👈"
                }
            
        
    
    
         GET | /check/{word}
        word - string
        Returns if the word exists

         Example 
        Url:  https://api.dextero.site/api/v1/check/arc       
        Response:  
            {
                "wordExists": true
            }
        
    

    
         GET | /definition/{word}
        word - string
        Returns the definition of a word

         Example 
        Url:  https://api.dextero.site/api/v1/definition/altar     
        Response: 
            {
                "word": "altar",
                "definition": [
                    "Parte a bisericii, despărțită de naos prin catapeteasmă, în care se oficiază liturghia.",
                    "Ridicătură de piatră, pământ sau lemn pe care, în antichitate, se aduceau jertfe zeilor."
                ]
            }
            
    

    
         GET | /length/{length}
        length - number
        Returns all the words matching this exact number of letters
    
         Example 
        Url:  https://api.dextero.site/api/v1/dextero/length/4    
        Response: 
            {
                "length": 1901,
                "words": [
                    "abac",
                    "abia",
                    "abil",
                    "abis",
                    "abur",
                    ...
                    "cert",
                    "culm",
                    "cost"
                ]
            }
        
    

    
        EXPERIMENTAL
         GET | /synonym/{word}
        word - string
        Returns all the synonims for the desired word
    
         Example 
        Url:  https://api.dextero.site/api/v1/dextero/synonym/cal     
        Response: 
            [
                {
                    "word": "cal",
                    "synonyms": [
                        "cabalină",
                        "mărţână",
                        "cal-de-mare",
                        "căluţ-de-mare",
                        "căluşel",
                        "război",
                        "scaun"
                    ]
                }
            ]
        
    

    
         GET | /contains
        Returns all the words containing the indicated sequence 
    
         Example 
        Url:  https://api.dextero.site/api/v1/dextero/contains/altar        
        Response: 
             
                {
                    "length": 7,
                    "words": [
                        "asfaltare",
                        "asaltare",
                        "coaltar",
                        "altar",
                        "cobaltare",
                        "baltard",
                        "altariu"
                    ]
                }
            
    

    
         GET | /random
        Returns a random word from the dictionary
    
         Example 
        Url:  https://api.dextero.site/api/v1/dextero/random        
        Response: 
             androni
